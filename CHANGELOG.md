# Changelog

## 0.1.2 — 2026-07-12

- 加入“蒸馏全行业TOP1，打开0到1000w赚钱之路”的首屏钩子；
- 将“获客 + 成交 = 赚钱”升级为第一视觉核心；
- 面向任何拥有能力、经验、产品、资源或想法的人，不再使用窄职业标签定义受众；
- 用餐饮菜单设计的第一笔订单作为首个公开示例；
- 将原窄角色案例替换为亲子科学工作坊多方场景，并同步测试、参考资料与Agent展示文案；
- 新增全仓检查，防止被删除的窄定位词重新出现。

## 0.1.1 — 2026-07-12

- 将核心定位收敛为“让你正在做、已经会做的事情，开始赚钱”；
- 把“获客 + 成交 = 赚钱”与 Nathan 的 ABCD 放到首屏；
- 默认先回答卖什么、卖给谁、怎么获客、怎么推进成交；
- 简单直销保持简单，多方角色和四流分析退到必要时调用；
- 更新 Skill 触发描述与 Agent 展示文案。

## 0.1.0 — 2026-07-12

- 发布单一 `niubiskill` 执行包；
- 建立角色与价值、钱、权力、信息四流模型；
- 保留 Acquisition / Bargaining / Customer Care / Development 原始 ABCD；
- 固定“一链、一环、一诺、一入口、一周”七天承诺实验；
- 加入持续价值测试、证据状态和 Economics / Permission / Risk 闸门；
- 加入三个跨行业合成案例、六个测试场景和自动仓库校验；
- 明确 clean-room 来源、Apache-2.0 许可与品牌边界。
