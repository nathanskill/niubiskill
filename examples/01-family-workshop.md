# 合成案例 01｜亲子科学工作坊

> `SYNTHETIC / 合成`：本案例由虚构角色和跨行业结构组成，不对应任何真实客户、收入或项目。

## 输入

> 小唐会设计亲子科学活动，社区中心可以提供场地，兼职老师负责现场交付。孩子参加，家长付款，场地方需要批准，但现在还没有试办规则。小唐下周应该先做什么？

# NIUBI Skill

## A. 商业闭环卡

### 一句话判断

这不是先做海报或收报名费的问题，而是先让场地方对一个范围清楚、安全责任明确的最小试办作出书面决定。

### 信息状态

- 全部业务设定：`SYNTHETIC / 合成`
- 小唐的活动设计能力：`用户自述（案例设定）`
- 场地可用时间、保险和安全要求：`待核`
- 家长需求、价格接受度和老师排期：`假设`

### 一链：最小角色链

`孩子（使用）↔ 家长（决策与付款）↔ 小唐（组织）↔ 老师（交付）↔ 社区中心（场地与批准）`

### 四流

- **价值流：** 孩子获得活动体验；家长获得清楚、安全、可预期的安排；老师完成交付；社区中心提高场地使用价值；小唐负责产品设计、组织和结果协调。
- **钱流：** 家长未来可能支付报名费，再按公开规则覆盖场地、老师、材料和组织成本；价格、退款与分配触发点均待核。
- **权力 / 决策与权限流：** 社区中心控制场地和安全规则；家长决定是否报名并提供同意；小唐只在获批范围内设计价格和流程；老师控制专业交付边界。
- **信息流：** 场地方提供规则；家长提供必要的年龄、同意和安全信息；老师反馈课程可行性；小唐只收集完成活动所需的最少信息。

### 用户主位置与价值捕获依据

小唐是活动组织者。收费依据不是“认识场地方和老师”，而是把活动设计、招生说明、现场交付和风险责任组织成一个可验收产品，并按公开规则分配收入和成本。

### 持续价值测试

即使家长、老师和场地方可以直接沟通，小唐仍需持续提供产品设计、排期协调、材料准备、报名治理和结果复盘；如果这些工作不存在，中间位置就没有独立价值。

### Gate与当前位置

`Amber`。场地、安全、保险和未成年人同意范围尚未确认，本周只推进书面审批，不招募、不收费，也不强行标记ABCD。取得可执行的批准条件后，再进入 `A — Acquisition` 验证家长需求。

### 一环：唯一瓶颈关系

`小唐 → 社区中心`

## B. 七天承诺实验

### 待证伪假设

如果小唐把人数、时长、人员、安全和停止条件压缩成一个最小试办，社区中心会在七天内作出批准、修改一个条件或暂停的书面决定。

### 一诺：本周有成本承诺

社区中心书面确认一次“8组家庭、90分钟、1名具名老师”的试办时间，以及必须遵守的安全和场地条件。

### 一入口

社区中心现有场地申请邮箱或正式预约入口。

### 唯一行动与行动资产

```text
主题：申请一次8组家庭的90分钟亲子科学活动试办

范围：8组家庭、90分钟、1名具名老师；未获批准前不公开招生、不收费。

希望确认：可用日期与房间、人数上限、保险/安全要求、材料限制、现场责任人和停止条件。

我们提供：活动流程、材料清单、老师资料、家长同意说明和活动后复盘。

请在[日期]前选择：批准 / 修改一个条件 / 暂停。
```

### 七天实验与证据

- 第1天：从正式入口提交一页试办申请。
- 第2–4天：只补充会影响批准的信息。
- 第5天：要求一个带日期的明确决定。
- 第7天：保存批准、修改或暂停的书面记录。

只有书面时间、条件和责任边界才算前进；口头支持不算试办已经成立。

### 样本/成本上限与停止条件

只申请一个场地、一个时间、一次试办；准备时间上限3小时。场地、安全、保险或未成年人同意范围不清时，不招生、不收费。

### Economics / Permission / Risk

- **Economics：** 场地、老师、材料、保险和组织工时待核；没有成本底线前不承诺价格。
- **Permission：** 未获场地书面批准和必要同意前，不发布招募或收集非必要信息。
- **Risk：** 任何无法满足的安全要求、责任不清或超出老师能力的活动内容都会触发停止。

### 第7天：继续、修改一个变量或停止

- 场地、时间和安全条件齐全：进入A，验证家长需求；
- 只缺一个可解决条件：只修改该条件；
- 权限或安全责任无法确认：停止该场地方案。

### 今天完成的一件事

把一页试办申请提交到社区中心的正式入口。
